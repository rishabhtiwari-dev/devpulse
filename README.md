# DevPulse

**Live Demo:** https://devpulse-seven-livid.vercel.app

A GitHub profile analyzer that visualizes developer activity and generates AI-powered insights.

## Features

- **Dashboard** — repos, stars, followers, top languages overview
- **Activity Heatmap** — contribution heatmap across the past year
- **Language Chart** — breakdown of languages used across all repos
- **AI Insights** — 3 personalized insights generated by Llama 3 (via Groq) based on commit patterns, language usage, and repo activity

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS  
**Backend:** Node.js, Express  
**AI:** Groq API (Llama 3)  
**Data:** GitHub REST API

## Run Locally

**Backend**
```bash
cd backend
npm install
# create .env with GROQ_API_KEY and GITHUB_TOKEN
npm run dev
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173`, enter any GitHub username and hit Analyze.

## Environment Variables

```
GROQ_API_KEY=your_groq_api_key
GITHUB_TOKEN=your_github_token
PORT=5000
```
